# 我们的圈子

> 一个给兄弟姐妹家里小朋友使用的微信小程序朋友圈

## 项目简介

“我们的圈子”是一个面向家庭内部场景的儿童轻社交小程序。小朋友可以创建自己的身份，使用语音、文字、照片和画画发布动态，也可以互相点赞、评论、查看消息通知和个人成就。

项目当前已经具备完整的核心体验，数据层使用微信云开发，适合继续做体验打磨、账号隔离和能力扩展。

## 当前状态

**当前阶段：核心功能已完成，正在继续完善体验和数据模型**

## 已实现功能

- 多身份管理：支持创建、编辑、删除孩子身份，支持 emoji 头像和自定义头像
- 动态发布：支持语音、文字、拍照、画板四种发动态方式
- 动态流：首页按时间倒序展示动态，支持下拉刷新
- 互动能力：支持点赞、评论、长按快速 emoji 回应
- 消息通知：新动态、点赞、评论都会触发通知，并展示未读红点
- 个人主页：展示个人动态数、获赞数和成就进度
- 成就系统：内置 10 个成长成就，根据发帖、评论、获赞等行为解锁
- 家长旁观模式：可进入只看不发的浏览模式
- 删除与重置：支持通过家长验证码删除动态和重置数据

## 待完善

- 微信登录和用户体系接入
- 动态分享能力
- 更多成就类型和成长激励机制

## 技术栈

- 微信小程序原生
- JavaScript + WXML + WXSS
- 微信云开发
- 云数据库：`children`、`posts`、`comments`、`notifications`
- 云存储：图片、语音等文件上传
- 云函数：初始化数据库、获取当前用户 `openid`

## 运行方式

1. 使用微信开发者工具打开项目根目录 `our-circle`
2. 在微信开发者工具中配置小程序 `AppID`
3. 确认已开通并绑定云开发环境
4. 如首次运行，先部署云函数并初始化数据库集合
5. 编译并运行小程序

## 云开发说明

项目依赖微信云开发能力，代码中已经初始化云环境，并通过统一数据层访问数据库和存储。

建议至少创建以下集合：

- `children`
- `posts`
- `comments`
- `notifications`

可使用 `cloudfunctions/init-db` 云函数辅助初始化集合。

## 核心使用流程

- 创建身份：首次进入后创建孩子身份
- 切换身份：首页顶部可切换不同孩子身份
- 发动态：点击右下角加号，选择语音、文字、拍照或画画
- 互动：可点赞、评论，或长按动态快速发送 emoji 回应
- 查看消息：在“消息”页查看新动态、点赞和评论通知
- 查看成长：在“我的”页查看统计和成就解锁情况

## 说明

- `project.private.config.json` 为本地微信开发者工具私有配置，不应提交到仓库
- 当前项目更接近单家庭内部使用场景，若要正式上线，建议补充用户体系、权限控制和数据隔离设计
