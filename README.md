## 如何使用

1. 用**微信开发者工具**打开 `our-circle` 目录
2. AppID 处填入你的小程序 AppID（或使用测试号）
3. 编译运行

### 核心操作

- **切换身份**：点击顶部头像区域 → 选择/编辑身份
- **发动态**：点击右下角「＋」→ 选择类型 → 录音/写字/拍照/画画
- **互动**：❤️点赞、💬评论、长按快速emoji回应
- **语音播放**：点击绿色语音气泡 → 声波动画+播放
- **家长模式**：点击顶部👁图标 → 只看不发
- **成就**：在「我的」页查看成就进度
- **重置数据**：我的 → 重置数据（恢复初始状态）
